# Nexus 2 — deactivated experiment

Nexus 2 is preserved here and is not loaded by Aurora.

Token measurements showed that compact JSON Family is nearly as efficient while remaining safer, standard, easier to validate, and less error-prone. The experiment still contains useful ideas, especially the CLI-like `--argument value` notation, and may inform a future protocol or visual representation.

The relay, delivery recovery, provider-driver, endpoint, and managed-surface improvements discovered during this experiment remain active. Only the Nexus 2 protocol layer is deactivated. The older `src/nexus` subsystem is a separate component and remains active and untouched.

Authoritative cloud tool language: **JSON Family**.
