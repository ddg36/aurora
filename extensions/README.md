# Aurora Extensions

Extensiones MV3 absorbidas por Aurora v2.

## Importadas

- `ash`: side panel, popup, content scripts, DNR y bridge browser.
- `bold`: popup y content script de foco visual.
- `browsernebula`: side panel/popup para control de navegador.
- `orion`: parser central como background MV3.
- `pcp4`: popup/content script para captura de pagina.
- `ytnoambient`: content script para YouTube.

## Pendiente

- `au-rtxnvidia`: no es extension MV3; va a modulos UI/backend de Aurora.
- `au-ytsounds`: utilidad Python minima; falta clasificar destino final.
